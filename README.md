## Installation

```bash
npm install
```

## Running locally

```bash
npm start
```
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## Running in docker (local dev)
```bash
bash run-docker-dev.sh
```
Open [http://localhost:8012](http://localhost:8012) to view it in the browser.

## Running in docker
```bash
bash run-docker.sh
```
